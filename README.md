# 📈 Smart Data Audit Pro
> **De datos caóticos a decisiones estratégicas en segundos.**

Este proyecto es una herramienta de **Auditoría de Datos** diseñada para transformar archivos Excel desorganizados en reportes limpios, visuales y listos para la toma de decisiones. Olvida las horas de limpieza manual; deja que la lógica lo haga por ti.



## ✨ ¿Qué hace esta herramienta?

* **Limpieza Automática:** Corrige formatos de texto, elimina espacios innecesarios y normaliza encabezados al instante.
* **Modo "Datos Perfectos":** Un filtro inteligente que elimina filas con errores, precios en cero o información faltante con un solo clic.
* **Auditoría de Anomalías:** Identifica y separa registros problemáticos para que sepas exactamente qué corregir en tu base de datos original.
* **Visualización Dinámica:** Genera gráficos de inversión y participación de mercado en tiempo real según los filtros aplicados.
* **Exportación Profesional:** Descarga el resultado final en un nuevo archivo Excel perfectamente estructurado.

## 🛠️ Tecnologías utilizadas

* **Python:** El motor de la lógica.
* **Streamlit:** Para la interfaz de usuario moderna y fluida.
* **Pandas:** Para el procesamiento avanzado de datos y limpieza.
* **Altair:** Para las visualizaciones interactivas.


## 🚀 Instalación y Uso Local

1. Clona este repositorio:
   git clone https://github.com/luislechuga55/data-cleaner-pro.git

2. Instala las dependencias:
    pip install -r requirements.txt

3. Ejecuta la aplicación:
    streamlit run app.py
